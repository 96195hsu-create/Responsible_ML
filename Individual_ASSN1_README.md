## Project Overview
This project focuses on preparing a dataset for analysis.
Starting from raw data, the notebook walks through the full process of cleaning, transforming, and organizing the data into a structured format that can be used for further work.

## What This Project Does
The workflow includes:
- Loading the dataset  
- Selecting relevant variables  
- Cleaning and filtering the data  
- Converting variables into appropriate formats  
- Creating new variables  
- Reviewing the final dataset  

## Tools Used
- pandas  
- numpy  

## How to Run
1. Open the notebook  
2. Run the cells from top to bottom  

The dataset is loaded from a public link, so no additional setup is needed.
